# Machine-Learning-Visa-Project







### Database used:

- MongoDB: https://account.mongodb.com/account/login


### Data link:

- Kaggle: https://www.kaggle.com/datasets/moro23/easyvisa-dataset



## How to run?

```bash
git clone 
```

```bash
conda create -n visa python=3.8 -y
```

```bash
conda activate visa
```

```bash
pip install -r requirements.txt
```
